## 如何自己建置？
1. 安裝 [NodeJS](https://nodejs.org)
2. 克隆此倉庫
```bash
 git clone https://github.com/taivs1106219/mymood_psy
 cd mymood_psy
```
3. 安裝依賴項
```bash
npm install
```
4. 建置
```bash
node scripts\builder.js
```
之後就可以在```out```資料夾看到建置完的安裝包啦~
